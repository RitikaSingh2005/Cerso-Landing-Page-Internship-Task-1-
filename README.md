# Cerso - Web Development Landing Page

![Cerso Hero Preview](./images/hero.png)

A fully responsive, modern, and professional landing page for **Cerso**, a web development company. This project was built as part of an internship task, focusing on clean design, performance, and user experience.

## 🚀 Features

- **Fully Responsive Design**: Scales flawlessly across mobile, tablet, and desktop viewports.
- **Modern Typography & UI**: Uses clean Google Fonts (Inter) with a curated Navy Blue (`#0d1b2e`) brand color palette.
- **Subtle Animations**: Implements Intersection Observers for smooth, scroll-triggered entrance animations.
- **Sticky Navigation**: A mobile-friendly navbar that adapts elegantly as the user scrolls.
- **Interactive UI Elements**: Custom hover states, glassmorphism effects, and highly readable cards.
- **Production Ready CSS**: Styled using Tailwind CSS, compiled and minified for optimal performance.

## 🧱 Sections Included

1. **Hero Section**: High-impact introduction with an engaging tagline and CTAs.
2. **About Us**: Information about Cerso's mission to serve small and growing businesses.
3. **Services**: Visual cards highlighting core offerings (Web Development, Management Systems, etc).
4. **Why Choose Cerso**: Key value propositions emphasizing affordability and modern design.
5. **Our Process**: A step-by-step visual workflow (Discuss ➔ Design ➔ Develop ➔ Deliver).
6. **Testimonials**: Social proof cards demonstrating client success.
7. **Contact / Get a Quote**: A sleek intake form with customized input fields and dropdowns.
8. **Footer**: Clean structural links, social media icons, and branding.

## 🛠️ Tech Stack

- **HTML5**: Semantic and accessible markup.
- **Tailwind CSS (v3)**: Utility-first CSS framework for rapid UI development and styling.
- **Vanilla JavaScript**: Lightweight DOM manipulation for mobile menus and scroll intersection observers (No heavy frameworks).
- **Phosphor Icons**: A flexible icon family for modern interfaces.

## 💻 Running Locally

To run this project locally on your machine:

1. Clone this repository:
   ```bash
   git clone https://github.com/RitikaSingh2005/Cerso-Landing-Page-Internship-Task-1-.git
   ```
2. Navigate into the directory:
   ```bash
   cd Cerso-Landing-Page-Internship-Task-1-
   ```
3. Open `index.html` directly in your browser, or start a local server using npx:
   ```bash
   npx serve .
   ```

## 🌐 Deployment

This project is built to be deployed easily on platforms like Vercel, Netlify, or GitHub Pages. Since it uses static HTML/CSS/JS, simply connect your repository to Vercel and it will deploy instantly with zero configuration needed.
